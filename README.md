# KZNERA Integrated Awareness Programme Website

## Overview of 22 Programmes
The KZNERA Integrated Awareness Programme encompasses 22 distinct programmes aimed at fostering community engagement and awareness. Each programme is designed to address specific needs and challenges faced by the communities in KwaZulu-Natal.

## Programme Statistics
- **Participants**: 1,065+
- **Budget**: R2.2M
- **Partners**: 50+

## Strategic Units
The programmes are structured under three strategic units:
1. **BI/Transformation Fund**: Focuses on fostering innovation and transformation within the community.
2. **Social Responsibility**: Enhances community engagement and social welfare initiatives.
3. **Communications**: Aims to raise awareness and promote transparency.

## Deployment Instructions for GitHub Pages
1. Push the code to the main branch of your GitHub repository.
2. Navigate to the repository settings.
3. Under the "Pages" section, select the `main` branch as the source.
4. Save the changes and visit the URL provided to access your live site.
